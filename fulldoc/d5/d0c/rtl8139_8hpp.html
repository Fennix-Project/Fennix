RcWRAP 

The WRAP bit is used to tell the NIC to wrap around the ring buffer when it reaches the end of the buffer.

Note
If this bit is set, the buffer must have an additional 1500 bytes of space at the end of the buffer to prevent the NIC from overflowing the buffer.
