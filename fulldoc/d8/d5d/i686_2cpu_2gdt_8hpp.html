◆ DC

uint8_t DC

Direction bit for data segments, conforming bit for code segments.

For data segments, this bit must be 1 for the segment to grow up (higher addresses).

For code segments, this bit must be 1 for code in the segment to be able to be executed from an equal or lower privilege level.
