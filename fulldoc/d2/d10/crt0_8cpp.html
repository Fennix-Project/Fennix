◆ DriverProbe()

int DriverProbe ( )

This is the first function that is called when the driver is loaded.

This function is to test if the driver is compatible with the hardware. Example: Like if there is a PCI device that the driver is for, or a CPU feature that etc.

Return 0 if the driver is compatible with the hardware. Otherwise, return a value that is not 0.

Note: In this function you cannot use variables that have constructors or destructors. Before DriverEntry, the constructors are called and after DriverFinalize, the destructors are called.
