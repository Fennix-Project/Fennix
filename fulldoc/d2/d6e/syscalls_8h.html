SYSCALL_ACCESS_DENIED 

Access denied This error code is returned when the current thread does not have the required permissions to perform the requested operation.

SYSCALL_INVALID_ARGUMENT 

Invalid argument This error code is returned when an invalid argument is passed to a syscall.

SYSCALL_INVALID_SYSCALL 

Invalid syscall This error code is returned when an invalid syscall number is passed to the syscall handler.

SYSCALL_INTERNAL_ERROR 

Internal error This error code is returned when an internal error occurs in the syscall handler.

SYSCALL_NOT_IMPLEMENTED 

Not implemented This error code is returned when a syscall is not implemented.

SYSCALL_ERROR 

Generic error This error code is returned when a syscall fails for an unknown reason.

SYSCALL_OK 

Success This error code is returned when a syscall succeeds.
