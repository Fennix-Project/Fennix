15  * KEY-1 VALUE-1 VALUE-1 leave file alone
16  * KEY-2 VALUE-2A VALUE-2B write VALUE-2B into file
17  * KEY-3 - VALUE-3 write VALUE-3 into file
18  * KEY-4 VALUE-4 - write an empty file
