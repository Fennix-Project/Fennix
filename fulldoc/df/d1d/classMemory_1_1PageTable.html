template<typename T >
T Memory::PageTable::Get ( T  Address)
inline

Get the Physical Address of a virtual address.

This function will return the physical address of a virtual address and not the virtual address of the current page table. This is intentional because the kernel page table has 1:1 mapping for the free memory.
